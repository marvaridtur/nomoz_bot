# 🕌 Namoz Tracker Bot

Kunlik 5 vaqt namozni kuzatib boruvchi Telegram bot.

## Buyruqlar

| Buyruq | Natija |
|--------|--------|
| `/bugun` | Bugungi namozlarni ko'rish va belgilash |
| `/statistika` | Oylik hisobot |
| `/eslatma` | Kunlik eslatmani yoqish / o'chirish |

## O'rnatish

```bash
pip install -r requirements.txt
python namoz_bot.py
```
